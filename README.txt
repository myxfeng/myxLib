#myxLibs :

包含了android 常用的架构库:
mvp rxjava  fresco okhttp3  retofit  BindView
本库都对以上架构进行了二次封装

日常开发的一些工具类:
ACache  :文件缓存
Base64
CheckUtils :list String检查工具
ClickFilter : 防止连点
Futils : 各种版本号 屏幕大小 网络 等工具集合
IntentUtils : 跳转工具集合
PreferenceUtils : 存储工具
RsaEncoder  : rsa 加密解密
StatusBarCompat : md 风格 状态栏
ToastUtils   : Toast

自定义view:
 ScrollViewPager,VerticalViewPager
